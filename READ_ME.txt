Добрый день.

1. Установить npm i
2. Вставить свой gmail && password в файл app.js 
3. Запустить app.js

-- В console увидете сообщения типа - 
        `Timeout 5s - if you have high speed internet, you can change the timeout setting, str(182)`

Работал с мобильного интернета, по этому ставил Timeout с запасом. Если у Вас скоростной интернет, то можете изминить Timeout.
В сообщениях указанно на кокой строке находиться Timeout (p.s.  возможно будет на пару строк смещение, работаю с Prettier).

В коде постарался прокоментировать дейтвия, при этом постарался не устраивать сочинение "Аля одуванчики под цветущей вишней..."

Здесь же постараюсь описать с какими проблемами встретился в тестовом задании (надеюсь на Ваш фидбек и коментарии).

1. К сожелению google блочит работу если Вы используете chromium или другие обертки браузеров,  которые служать для выполнеия тестовых работ. На старте повозился с этим, потом нашел решение запуска через Playwright && firefox.

2. Следующая загвозка возникла, когда после логинизации я сразу пытался перейти по URL сразу с набором фильтров по типу - 
  https://mail.google.com/mail/..../#advanced-search/is_unread=true&isrefinement=true
	google постоянно перекидывал на строницу логиинизации... 
Если можно было решить вопрос в такой способ, оставьте пожалуста свой коментариий 

3. В "block - choose SETTINGS for change language - блок відкриття налаштувань" - этот блок создал для таго чтобы пробигаться по одному языку( на старте хотел сделать только для Eng && Ukr, но мы не знаем на каким языком пользоваеться пользователь...). По хорошему после оканчания скрипта , необходимо было бы вернуть настройки к исходному значению, но уже и так затянул с отсылкой тестого.
